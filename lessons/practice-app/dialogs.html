Dialogs & Windows

Practice Lab UI Automation

Practice handling browser dialogs (alert, confirm, prompt), new window / tab popups, file downloads, and security (basic-auth) alerts with Playwright.

1. Alert Dialog window.alert()

Triggers a browser alert(). Playwright: listen with page.on('dialog', d => d.accept()).

2. Confirm Dialog window.confirm()

Triggers a confirm() dialog. Playwright: d.accept() → OK, d.dismiss() → Cancel.

3. Prompt Dialog window.prompt()

Triggers a prompt()d.accept('your text') to type a value, or d.dismiss() to cancel.

4. New Window / Tab window.open()

Opens a new browser tab. Playwright: use page.context().waitForEvent('page') before clicking to capture the new page handle.

5. File Download download event

Triggers a file download. Playwright: use page.waitForEvent('download') before clicking, then download.saveAs(path).

6. Security Alert – Basic Auth HTTP Auth

Simulates a browser HTTP Basic-Auth challenge dialog. Playwright: use page.authenticate({ username, password }) before navigating, or handle via page.on('dialog', ...) for credential prompts.
Valid credentials: username admin, password secret.

7. Timed / Delayed Alert setTimeout

An alert fires after a 3-second delay. Playwright: register the page.on('dialog', ...) handler before clicking — the event fires whenever the dialog appears.
