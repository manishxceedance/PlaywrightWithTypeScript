Lesson 3: Functions and Object Types

Level: Beginner-Intermediate Learn how to write, type, and use functions and objects in TypeScript with clear, practical examples.

1. Functions: The Basics

A function is a reusable block of code. You give it input (parameters), it does something, and it can return a result.

// Add two numbers and return the result
function add(a: number, b: number): number {
  return a + b;
}
const sum = add(2, 3); // 5
console.log(sum);

2. Optional and Default Parameters

You can make a parameter optional (with ?) or give it a default value.

// 'cycle' has a default, 'owner' is optional
function buildReportTitle(project: string, cycle = "Sprint-1", owner?: string): string {
  const ownerText = owner ? ` by ${owner}` : "";
  return `${project} - ${cycle}${ownerText}`;
}
console.log(buildReportTitle("Payments")); // Payments - Sprint-1
console.log(buildReportTitle("Payments", "Sprint-2", "Asha")); // Payments - Sprint-2 by Asha

3. Function Type Aliases

Type aliases let you describe the shape of a function, so you can reuse it and make your code safer.

// Comparator is a function that takes two numbers and returns a boolean
type Comparator = (actual: number, expected: number) => boolean;
const isEqual: Comparator = (a, b) => a === b;
console.log(isEqual(5, 5)); // true

4. Typing Object Parameters

Use object types to make sure a function only accepts objects with the right properties.

// Define the shape of the input object
type ApiCheckInput = {
  endpoint: string;
  expectedStatus: number;
  requiresAuth: boolean;
};
function describeApiCheck(input: ApiCheckInput): string {
  const authTag = input.requiresAuth ? "Auth" : "NoAuth";
  return `${authTag}: ${input.endpoint} should return ${input.expectedStatus}`;
}
console.log(describeApiCheck({ endpoint: "/api/health", expectedStatus: 200, requiresAuth: true }));

5. Array of Objects Example

Arrays let you store lists of things. In TypeScript, you can make arrays of objects with a specific shape.

// Define the shape of a test case
type TestCase = {
  id: string;
  title: string;
  automated: boolean;
};
const testCases: TestCase[] = [
  { id: "TC-1", title: "Login success", automated: true },
  { id: "TC-2", title: "Forgot password", automated: false }
];
// Print only the automated test cases
testCases.forEach(tc => {
  if (tc.automated) {
    console.log(tc.id, tc.title);
  }
});

6. Function Overloading

Function overloading lets you write one function that works with different types or numbers of arguments, while keeping type safety.

// Overload signatures
function combine(a: string, b: string): string;
function combine(a: number, b: number): number;
// Implementation
function combine(a: any, b: any): any {
  return a + b;
}
const result1 = combine("Hello, ", "world!"); // string
const result2 = combine(2, 3); // number
