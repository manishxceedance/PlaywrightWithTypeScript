Lesson 6: Generics for Reusable Components

Level: Intermediate-Advanced Learn how to write reusable, type-safe code using TypeScript generics.

Generics let you write code that works with any type, while keeping type safety. They’re essential for building flexible utilities, data structures, and APIs.

1. What is a Generic?

A generic is a placeholder for a type. It lets you write one function, class, or type that works for many types.

// Generic function
function wrapInArray(value: T): T[] {
  return [value];
}
const numbers = wrapInArray(10);      // number[]
const names = wrapInArray("Neha");   // string[]

2. Generic Types and Interfaces

You can use generics in type aliases and interfaces to describe reusable shapes.

// Generic API response type
type ApiResponse = {
  status: number;
  data: T;
  error?: string;
};

type UserPayload = { id: string; name: string };
const response: ApiResponse = {
  status: 200,
  data: { id: "U1", name: "Anu" }
};

3. Generic Constraints

Use extends to require that a generic type has certain properties.

// Only accepts values with a length property
function printLength(value: T): void {
  console.log(value.length);
}
printLength([1, 2, 3]); // OK
printLength("hello");   // OK
// printLength(42);     // Error

4. Multiple Type Parameters & Defaults

You can use more than one generic, and provide default types.

// Pair of values
type Pair = { first: A; second: B };
const p1: Pair = { first: 1, second: true };
const p2: Pair = { first: 2, second: "default" };

5. Generic Classes and Interfaces

Classes and interfaces can be generic too.

// Generic class for a simple stack
class Stack {
  private items: T[] = [];
  push(item: T) { this.items.push(item); }
  pop(): T | undefined { return this.items.pop(); }
}
const numStack = new Stack();
numStack.push(1);
numStack.push(2);
console.log(numStack.pop());

6. Utility Generics

Many built-in utility types use generics (like Partial, Record, etc.).

// Partial makes all fields optional
type Bug = { id: string; summary: string };
const draft: Partial = { summary: "Crash" };

// Record creates a map
type TestResults = Record;
const results: TestResults = { "TC-1": "pass" };

7. Practical Utility: Test Data

Use generics to write flexible, type-safe utilities for test data.

// Get the first item of any array
function firstItem(items: T[]): T | undefined {
  return items[0];
}
const firstCase = firstItem(["TC-1", "TC-2"]);
