Module 15: Debugging

Beginner Friendly This module explains concepts in plain language and then gives practical examples.

What You Will Learn

How to investigate failures using debug tools and traces.

Simple Explanation

Debug mode lets you inspect test steps interactively. Trace viewer provides timeline, screenshots, and actions for root-cause analysis.

Real-World Example

A click failed because button was hidden by a popup. Trace reveals popup appeared before click step.

Step-by-Step Example (Code)

npx playwright test --debug
npx playwright show-trace trace.zip

Beginner Notes (Line by Line)

  1. npx playwright test --debug - This runs a Playwright command in your terminal.
  2. npx playwright show-trace trace.zip - This runs a Playwright command in your terminal.

Common Mistake to Avoid

Rerunning failed tests repeatedly without reading trace evidence.

Practice Exercise

  1. Intentionally fail one test and inspect trace timeline.
  2. Add screenshots/logs around critical checkpoints.
